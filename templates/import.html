Excel File Formatting Rules:
  • Sheet Names: Each sheet represents a class name (e.g., 'Nursery', 'Prep', 'One').
  • Row 3 (Header): Cell headers are expected on row index 2 (third row). Column 1 must represent 'Name', column 2 represents 'Father Name'.
  • Monthly Fee Column: Columns containing "month" (like "Month", "Month 26") and not representing specific months hold the tuition fee value.
  • Paid Columns: Columns starting with month contractions (like "Nov", "Dec", "Mar", "April") hold the actual payments made.
